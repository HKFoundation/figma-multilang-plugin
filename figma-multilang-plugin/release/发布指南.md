# Figma 插件发布指南

## 发布包位置

发布文件位于 `figma-multilang-plugin/release/` 目录，包含以下文件：

- `manifest.json` - 插件配置
- `code.js` - 插件主代码
- `ui.html` - 插件界面

## 发布到 Figma 插件商店

### 第一步：注册开发者账号

1. 访问 https://www.figma.com/communitydevelopers
2. 使用你的 Figma 账号登录
3. 进入开发者设置页面

### 第二步：创建插件

1. 点击 "Create new plugin"
2. 填写插件信息：
   - **Plugin name**: 多语言文本替换
   - **Description**: 从 Excel 导入翻译数据，批量替换 Figma 文本为多语言版本
   - **Category**: 建议选择 "Utilities" 或 "Design"

### 第三步：上传代码

1. 在插件设置页面，选择 "Upload to Figma" 选项
2. 将 `release/` 目录中的三个文件上传：
   - manifest.json
   - code.js
   - ui.html

### 第四步：准备图标

需要提供一个 128x128 像素的 PNG 图标：
- 尺寸：128 x 128 像素
- 格式：PNG（支持透明背景）
- 建议：简洁明了，体现插件功能

### 第五步：提交审核

1. 填写完整的插件商店信息
2. 上传插件图标
3. 撰写清晰的描述和使用说明
4. 点击提交

### 审核时间

- 初审通常需要 1-3 个工作日
- 如有需要补充的材料，Figma 会通过邮件联系你

## 本地测试

在正式发布前，建议先在本地测试：

1. 打开 Figma Desktop
2. 进入 Plugins → Development → Import plugin from manifest
3. 选择 `figma-multilang-plugin` 文件夹
4. 运行插件进行测试
