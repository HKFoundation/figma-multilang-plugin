多语言文本替换

A Figma plugin to replace text with multilingual translations.

Contact

Email: 510800230@qq.com

How to Install & Use

1. Download the plugin files (manifest.json, code.js, ui.html)
2. Open Figma Desktop app
3. Go to Plugins → Development → Import plugin from manifest
4. Select the folder containing manifest.json
5. Open any design file in Figma
6. Go to Plugins → Development → 多语言文本替换
7. The plugin is ready to use

Features

  • Import translations from Excel files
  • Support Chinese, Japanese, Spanish and more languages
  • Batch replace text in Figma designs
  • Create separate pages for each language
